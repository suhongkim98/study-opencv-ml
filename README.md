# study-opencv-ml
## opencv, ml, OpenPose 공부 레포지토리입니다.

### 가상환경 생성
```
virtualenv .venv
```

### virtualenv 가상환경 활성화
```
source .venv/bin/activate
```

### 의존성 설치
```
pip3 install jupyter
pip3 install opencv-python
pip3 install matplotlib
pip3 install sklearn
pip3 install seaborn
pip3 install tensorflow
pip3 install keras
```

### 주피터 노트북 열기
```
jupyter notebook
```

### virtualenv 비활성화
```
deactivate
```


### 의존성 확인
```
pip3 list
```
